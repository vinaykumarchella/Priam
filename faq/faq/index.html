    Increasing Cassandra cluster

    There are 3 ways to increase the Cassandra cluster capacity: 1. Instance Upgrade: With this you replace the instance types of the Cassandra cluster by more/less powerful instances, one by one. Note: This does not increase the number of coordinator nodes. 2. Doubling the cassandra cluster: This will double the number of instances in your Cassandra cluster. This is usually preferred way to go (if possible) as this truly doubles the capacity of the Cassandra cluster by providing more co-ordinates nodes and gives more JVM's. 3. Add a new region to existing Cassandra cluster.
